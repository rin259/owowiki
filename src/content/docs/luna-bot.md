## 重点插件与用法

### 帮助

- 列插件：`/help`
- 树形帮助：`/help -t`
- 看单个插件：`/help 插件名`
- 看单个插件树：`/help --tree 插件名`

### 云签到

- `签到`
- `积分`
- `挖矿`
- `我的背包`
- `钓鱼`
- `我的鱼篓`
- `出售 物品名`
- `功能`
- `功能 子项`
- `领取积分补助`
- `签到状态`
- `排行榜`
- `打劫@某人`
- `抽奖 数量`
- `转账 金额 @某人`
- `猜拳石头|剪刀|布 金额`
- `猜数字 数字`

建议先发 `功能` 看云端返回的完整菜单。

### 在线运行代码

格式是多行：

<p class="cmd-tip">可直接复制发送</p>

```text
code [语言] [stdin]
[代码]
```

示例：

<p class="cmd-tip">可直接复制发送</p>

```text
code py 你好
print(input())
```

支持的语言：`c/cpp/c#/py/php/go/java/js`。

### 禁漫下载

- 命令：`jm <禁漫号>`
- 例子：`jm 123456`
- 行为：先下载，再按配置发转发消息或上传文件

### 系统命令行

管理员专用。

- Shell：`/sh <命令>`
- 例子：`/sh echo Hello World`
- Shell 模式在 Windows 上禁用

- 进程命令：`/cmd <命令>`
- 例子：`/cmd echo Hello World`

### 再润

管理员专用，且需要 `@OwO` 或命令前缀命中。

- 停机：`stop`、`shutdown`、`停机`
- 重启：`restart`、`reboot`、`重启`

### 链接分享解析

- 直接发送支持平台的 `BV号`、链接、小程序或分享卡片即可
- 额外命令：
  - `bm <BV号> [页码]`：下载 B 站音频
  - `ym <YouTube链接>`：下载 YouTube 音频
  - `blogin`：B 站扫码登录
- 管理员命令：
  - `开启解析`
  - `关闭解析`

### Instagram 解析

- 命令：`ins <instagram 链接>`
- 别名：`instagram <链接>`
- 也支持直接发送含 `instagram.com/p|reel|stories/...` 的链接
- 超过 3 个媒体文件时会走合并转发

### 插件响应鉴权

这是管理员专用的黑名单插件。

- 用户维度：`#(全局|插件名称)(拉黑|解黑)(全员|用户id)`
- 群维度：`#(全局|插件名称)(封禁群|解禁群)(群号|留空封禁所在群)`

### 人生重开

- 触发：`@Bot remake`
- 别名：`liferestart`、`人生重开`、`人生重来`
- 快捷：`随机人生`
- 进入后会要求选择天赋；回复如 `0 1 2` 完成选择，也可发 `随机`

### 🦌管签到

这个插件已经在 Luna 里安装并启用，版本是 `nonebot-plugin-deer-pipe v0.6.2`。

<p class="cmd-tip">可直接复制发送；命令里的“🦌”也都可以换成“鹿”。</p>

#### 自己🦌一次

适合平时自己签到。

<p class="cmd-tip">可直接复制发送</p>

```text
🦌
```

说明：

- 表示自己今天 `🦌` 1 次
- 如果今天已经 `🦌` 过，再发一次会继续累加次数
- 机器人通常会回一张本月 `🦌` 日历图

#### 帮别人🦌一次

只能在群聊或频道里用，因为要 `@` 人。

<p class="cmd-tip">可直接复制发送</p>

```text
🦌 @某人
```

示例：

<p class="cmd-tip">可直接复制发送</p>

```text
🦌 @Asterism
```

说明：

- 这是“帮别人 `🦌` 1 次”
- 如果对方关闭了“帮🦌”，机器人会提示不能帮
- 如果对方被管理员临时禁 `🦌`，也会被拦下

#### 补🦌本月之前的某一天

用来补本月漏掉的日期。

<p class="cmd-tip">可直接复制发送</p>

```text
补🦌 3
```

说明：

- 表示补本月 `3` 号
- 只能补本月里、并且早于今天的日期
- 不能补未来日期
- 已经补过或已经 `🦌` 过的那天，不能重复补

#### 看自己的本月🦌日历

<p class="cmd-tip">可直接复制发送</p>

```text
🦌历
```

说明：

- 会返回你本月的 `🦌` 日历图
- 适合看哪天漏了、哪天 `🦌` 了几次

#### 看别人的本月🦌日历

只能在群聊或频道里用。

<p class="cmd-tip">可直接复制发送</p>

```text
🦌历 @某人
```

示例：

<p class="cmd-tip">可直接复制发送</p>

```text
🦌历 @Asterism
```

#### 看本群🦌排行榜

只能在群聊或频道里用。

<p class="cmd-tip">可直接复制发送</p>

```text
🦌榜
```

说明：

- 返回本月当前群聊里的 Top 5 `🦌` 榜
- 不是全局榜，只看当前群

#### 设置别人能不能帮你🦌

这个命令是用户自己控制的，不需要管理员。

<p class="cmd-tip">可直接复制发送</p>

```text
帮🦌 off
```

```text
帮🦌 on
```

说明：

- `帮🦌 off`：不允许别人帮你 `🦌`
- `帮🦌 on`：重新允许别人帮你 `🦌`
- 这个命令通常只能在群聊或频道里生效

#### 给别人设置“允许/禁止帮🦌”

群管理员专用。

<p class="cmd-tip">可直接复制发送</p>

```text
帮🦌 off @某人
```

```text
帮🦌 on @某人
```

示例：

<p class="cmd-tip">可直接复制发送</p>

```text
帮🦌 off @Asterism
```

说明：

- 只有群管理员才能对别人这样设置
- 普通成员发送时会提示权限不足

#### 临时禁🦌 / 解禁

群管理员专用。

<p class="cmd-tip">可直接复制发送</p>

```text
禁🦌 @某人 3d
```

```text
禁🦌 @某人 12h
```

```text
禁🦌 @某人
```

说明：

- `禁🦌 @某人 3d`：禁止对方接下来 `3` 天内 `🦌`
- `禁🦌 @某人 12h`：禁止对方接下来 `12` 小时内 `🦌`
- `禁🦌 @某人`：不带时长时，视为解禁
- 这个时长参数走 `pytimeparse` 风格，常见写法有：`30m`、`2h`、`3d`
- 插件内部限制最长大约 `30` 天，太长会被拒绝

#### 打开插件帮助

<p class="cmd-tip">可直接复制发送</p>

```text
🦌帮助
```

说明：

- 会返回插件内置帮助
- 如果你记不清命令，先发这个最省事
- `鹿帮助` 也可以

### 表情包制作

已确认的入口：

- `表情包制作`
- `表情详情 <表情名或关键词>`
- `表情搜索 <关键词>`
- `随机表情 <图片或文字>`
- `[我的][全局]<时间段>表情调用统计 [表情名]`

管理员专用入口：

- `禁用表情 <表情名>`
- `启用表情 <表情名>`
- `全局禁用表情 <表情名>`
- `全局启用表情 <表情名>`

实际制作时，发送 `关键词 + 图片/文字` 即可。
